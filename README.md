# CommitPush (커밋푸시)

**"당신의 작업을 커밋처럼 기록하고,
당신의 실행력을 시각화하는 시스템"**

CommitPush는 작업 내용을 "커밋" 단위로 기록하고 관리하는 메모 기반 시스템입니다.
단순 메모앱이 아니라, 생각과 업무를 버전관리하는 시스템을 지향합니다.

## 빠른 시작

개발 서버 실행:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.
